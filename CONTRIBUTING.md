## 大家注意，每次开发新功能的时候记得
```bash
git checkout dev #如果你本地有dev分支（如果没有就是checkout -b)
git pull origin dev

git checkout feature/** #同样和上面的考虑-b，

git merge dev #这个merge是发生在本地的
```
### 之后才能继续（原因是git是看分支之间有没有关系，这样能始终保持你的feature分支和远程dev是父子关系的，到时候才能merge到远程）
